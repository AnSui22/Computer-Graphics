## 빌드 방법
1. Cmake 설치
2. 코드 Clone 또는 Zip 파일 다운 후 압축 풀기
3. 다운받은 폴더 안에 build 폴더 생성
4. Cmake gui 사용
   - Where is the source code: 폴더명
   - Where is the build code: 폴더/build
5. Configure -> Generate
6. build 폴더에 들어가서 sln 파일 실행
7. 원하는 파일 시작프로젝트로 설정
8. 실행
